About ValoCheck

What is ValoCheck?

ValoCheck is a fast, lightweight Valorant stats tracker built for players who want instant access to their performance data without the bloat.

Look up any player by Riot ID, check K/D ratio, win rate, headshot percentage, agent performance, match history, and more.

Features

  • Player profiles with detailed competitive stats
  • MMR history tracking with visual rank progression
  • Regional leaderboards (EU, NA, AP, KR) updated every 6 hours
  • Agent meta insights with pick/win rates from top-ranked play
  • Crosshair builder with code export and pro player presets
  • Player comparison tool for side-by-side analysis
  • Game tips and guides for every rank level

Data Sources

Player data is sourced from the Henrik Dev API (unofficial Valorant API). Agent and map assets are from valorant-api.com.

ValoCheck is not endorsed by Riot Games and does not reflect the views or opinions of Riot Games or anyone officially involved in producing or managing Valorant.
